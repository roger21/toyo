### sources utilisées par les boutons « Random » des générateurs

#### notes :
- `smileys` est l'assemblage des smileys persos tels que dans le fichier [smileys.txt](https://github.com/roger21/toyo/blob/main/generateurs/_api/smileys.txt) et des smileys de base et étendus du forum tels que dans le fichier [scripts.js](https://github.com/roger21/toyo/blob/main/generateurs/scripts.js)
- pour la description des autres sources voir le fichier [README_files.md](https://github.com/roger21/toyo/blob/main/generateurs/_random/README_files.md)
- sauf spécifié, les boutons random remettent le ou les autres paramètres par défaut
