[b]Bulle[/b] [img]https://reho.st/https://generateurs.super-h.fr/bulle/?t=Vive%20les%20g%C3%A9n%C3%A9rateurs&delta=0&s=wkatepebaboune&police=carter&taille=1[/img]
[b]Paramètres :[/b]
[*]t : le contenu de la bulle
[*]delta : le décalage vertical de la bulle en pixels (de 0 à 100)
[*]flip (sans valeur) : met la bulle de l'autre côté
[*]police : la police d’écriture (cartoon, carter ou tintin)
[*]taille : la taille de la police (de 1 à 3)
[*]s (facultatif) : le code du smiley, annule l'effet du paramètre delta
[*]r (facultatif) : le rang du smiley si nécessaire (de 0 à 10)
[b]Bouton :[/b]
[*]random : choisit une citation (issue des signatures des forumeurs) au hasard, un smiley au hasard, la police Carter One et la taille 3 et remet les autres paramètres par défaut



[b]Moit-Moit[/b] [img]https://reho.st/https://generateurs.super-h.fr/moitmoit/?o&s1=%3Alol%3A&s2=dawa[/img]
[b]Paramètres :[/b]
[*]s1 : le code du premier smiley
[*]r1 (facultatif) : le rang du premier smiley si nécessaire (de 0 à 10)
[*]s2 : le code du deuxième smiley
[*]r1 (facultatif) : le rang du deuxième smiley si nécessaire (de 0 à 10)
[*]v (sans valeur) : découpe les smileys verticalement
[b]Boutons :[/b]
[*]switch : inverse les deux smileys
[*]random : choisit deux smileys au hasard et remet les autres paramètres par défaut

[b]Rofl[/b] [img]https://reho.st/https://generateurs.super-h.fr/rofl/?v=8&s=shay&t=2&m=3&dx=0[/img]

[b]Golden[/b] [i]par castormalin[/i] [img]https://reho.st/https://generateurs.super-h.fr/golden/?v=8&s=castormalin&r=4&stars=5&golden=40&av=B&ah=G&vert&taille=3&frames=20[/img]
[b]Paramètres :[/b]
[*]s : le code du smiley
[*]r (facultatif) : le rang du smiley si nécessaire (de 0 à 10)
[*]stars : le nombre maximal d'étoiles (de 0 à 100)
[*]golden : le taux de doré en % (de 0 à 100)
[*]wo (sans valeur) : ne met pas le texte "GOLDEN"
[*]av : l'alignement vertical du texte (H, haut ou B, bas)
[*]ah : l'alignement horizontal du texte (G, gauche ou D, droite)
[*]vert (sans valeur) : met le texte à la verticale
[*]taille : la taille du texte (de 1 à 3)
[*]frames : le nombre de frames avant que la sequence d'étoiles ne reboucle (de 1 à 100)
[*]v : la vitesse (de 1 à 10)
[b]Bouton :[/b]
[*]random : choisit un smiley au hasard et remet les autres paramètres par défaut

[b]Spin[/b] [i]par roger21[/i] [img]https://reho.st/https://generateurs.super-h.fr/spin/?v=8&s=roger21&r=3&mode=0&rayonx=200&rayony=100&angle=0&asteps=100&rsteps=10&branches=8&fixe[/img]
[b]Paramètres :[/b]
[*]s : le code du smiley
[*]r (facultatif) : le rang du smiley si nécessaire (de 0 à 10)
[*]mode : le mode de rotation (0, simple ou 1, en spirale)
[*]rayonx : le rayon horizontal en pixels (de 0 à 1000)
[*]rayony : le rayon vertical en pixels (de 0 à 1000)
[*]angle : l'angle initial du smiley et de sa position en degrés (de 0 à 359)
[*]asteps : le nombre de positions intermédiaires dans la rotation (de 0 à 100)
[*]rsteps : le nombre de positions intermédiaires dans la spirale (de 0 à 100)
[*]branches : le nombre de branches (de 0 à 100)
[*]rofl (sans valeur) : fait tourner les smileys sur eux-mêmes
[*]fixe (sans valeur) : garde les smileys fixes au lieu de suivre l'angle de rotation
[*]center (sans valeur) : rajoute un smiley au centre
[*]v : la vitesse (de 1 à 10)
[b]Bouton :[/b]
[*]random : choisit un smiley au hasard et remet les autres paramètres par défaut
